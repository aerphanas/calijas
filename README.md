# calijas

simple calculator cli

## available operation

- "+" for addition
- "−" for subtraction
- "×" for multiplication
- "÷" for division

## how to run

- download package from release page  
[Release](https://github.com/aerphanas/calijas/releases/tag/v0.0.1)

- run with jre

```sh
java -jar calijas-v0.0.1.jar -x 1 -y 1 -o + # 2
java -jar calijas-v0.0.1.jar -x 60 -y 9 -o + # 69
```
